##  Описание проекта

**OnlineSale** — полнофункциональный интернет-магазин, разработанный в рамках учебной практики и выпускной квалификационной работы.

###  Цель проекта
За 4 недели разработать и задокументировать информационную систему онлайн-продаж с полным циклом: от каталога товаров до оформления заказа и оплаты.

---

## Функциональность

###  Реализовано

| Модуль | Статус | Описание |
|--------|--------|----------|
| **Каталог товаров** |  Готово | Просмотр товаров с категориями, карточки товаров с фото |
| **Аутентификация** |  Готово | Регистрация, вход, выход, личный кабинет |
| **Корзина** |  Готово | Добавление/удаление, хранение в сессии, подсчёт суммы |
| **Оформление заказа** | Готово | Контактные данные, адрес доставки, выбор способа оплаты |
| **Способы оплаты** |  Готово | Карта, СБП, OnlineSale Wallet (скидка 10%) |
| **История заказов** |  Готово | Просмотр всех заказов с статусами |
| **База данных** |  Готово | Внутренний кошелёк, пополнение, оплата со скидкой |
|  **Админ-панель** |  Готово | Реализованы и работают все основные модели:User (пользователи с ролями) Category (категории товаров) Product (товары с фото) CartItem (позиции корзины) Order (заказы) OrderItem (позиции заказа) Wallet (кошелёк пользователя) |

### В процессе реализации

| Модуль | Приоритет | Описание |
|--------|-----------|----------|
|  **Фильтрация и поиск** | Высокий | Фильтр по категориям, цене; поиск по названию |
|  **Пагинация товаров** | Низкий | Постраничная навигация в каталоге |
|  **Панель менеджера** | Высокий |Список заказов с фильтром по статусу, изменение статуса |
|  **Логирование действий** | Средний | Модель ActionLog (пользователь, действие, дата, объект) |
|  **Экспорт заказов** | Средний |Выгрузка в CSV (эндпоинт /admin/orders/export/) |
|  **Настройка Django Admin** | Низкий | list_display, search_fields, фильтры для всех моделей |
|  **Пояснительная записка** | Низкий | 50-70 страниц по ГОСТ (Times New Roman, 14 кегль, 1.5 интервал) |

Установка и запуск
Требования
Python 3.10 или выше
pip (менеджер пакетов Python)
Пошаговая инструкция

Клонируйте репозиторий
git clone https://github.com/..../DiplomaShop.git
cd DiplomaShop

Создайте виртуальное окружение
python -m venv venv

Активируйте виртуальное окружение
Windows:
venv\Scripts\activate

macOS/Linux:
source venv/bin/activate

Установите зависимости
pip install -r requirements.txt

Примените миграции
python manage.py migrate

Создайте суперпользователя (администратора)
python manage.py createsuperuser

Запустите сервер разработки
python manage.py runserver

Откройте браузер
Главная страница: http://127.0.0.1:8000/
Админ-панель: http://127.0.0.1:8000/admin/
